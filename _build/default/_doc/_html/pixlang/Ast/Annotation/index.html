Module Ast.Annotation

Module representing annotations of elements of the program. It contains the Util.Position.t of the parsed text that represent the element, and its type_expression. The type is only useful for expressions.

type t

The Annotation type.

val create : Util.Position.t -> t

create pos creates an t with the position set as pos and no type set. To be used within the parser (when creating the Ast), and if new expressions/statement are created during the analysis phase (notably if implementing the implicit cast).

val get_pos : t -> Util.Position.t

get_pos annotation return the position within annotation.

val get_type : t -> type_expression option

get_type annotation returns Some t if t is the type_expression within the annotation, and None if none is set.

val set_type : t -> type_expression -> unit

set_type annotation t sets the type within annotation to be t
